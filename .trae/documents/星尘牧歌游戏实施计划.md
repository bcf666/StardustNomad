# 《星尘牧歌》游戏设计实施计划

## 一、项目概述

**项目名称**: StardustNomad / 星尘牧歌
**项目类型**: 太空沙盒建造游戏
**视觉风格**: 体素/Minecraft风格
**技术方案**: Three.js 3D渲染 + 原生JavaScript游戏引擎
**目标**: 实现完整设计文档中的所有核心系统

## 二、技术架构

### 2.1 技术栈
- **渲染引擎**: Three.js (WebGL)
- **体素系统**: 简化版体素引擎，支持大规模世界
- **构建工具**: 纯前端静态资源，无需构建
- **存储**: LocalStorage (存档) + IndexedDB (大型数据)

### 2.2 核心模块划分

```
┌─────────────────────────────────────────────────────────┐
│                      游戏主循环                          │
│                   (requestAnimationFrame)               │
└─────────────────────────────────────────────────────────┘
         │           │           │           │
         ▼           ▼           ▼           ▼
┌─────────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│  渲染系统   │ │ 物理系统  │ │ 飞船系统  │ │ UI系统   │
```

## 三、游戏系统详细规划

### 3.1 体素渲染引擎

**文件**: `src/engine/voxel.js`

核心功能:
- 体素块类型定义 (16种基础方块)
- Chunk系统 (16x16x16区块)
- 动态加载/卸载 (玩家周围8个Chunk)
- 面的可见性裁剪 (仅渲染可见面)
- 材质系统 (程序化贴图)

```javascript
BlockType = {
  AIR: 0, STONE: 1, DIRT: 2, GRASS: 3, IRON_ORE: 4,
  COPPER_ORE: 5, SILICON: 6, ICE: 7, LAVA: 8,
  WATER: 9, SAND: 10, WOOD: 11, LEAVES: 12,
  GLASS: 13, METAL: 14, SCI_FI_PANEL: 15
}
```

### 3.2 程序化星球生成

**文件**: `src/world/planet-generator.js`

功能:
- 噪声函数 (Simplex noise用于地形)
- 星球种子系统 (可复现)
- 星球类型生成:
  - 岩石星球 (高地形变化)
  - 冰冻星球 (冰雪方块)
  - 熔岩星球 (熔岩池)
  - 海洋星球 (浅水覆盖)
  - 沙漠星球 (沙丘)
- 资源分布算法 (矿物节点)
- 生物群落 (根据温度/湿度)

**文件**: `src/world/galaxy-generator.js`

功能:
- 恒星系生成 (1-4颗行星)
- 恒星类型 (红/黄/蓝矮星)
- 星际坐标系统
- 虫洞/异常点生成

### 3.3 飞船系统

**文件**: `src/ship/ship.js`
**文件**: `src/ship/ship-builder.js`
**文件**: `src/ship/physics.js`

核心功能:
- 飞船数据结构 (网格位置 + 方块类型)
- 飞行物理:
  - 推力 = 引擎方块数量 × 引擎功率
  - 质量 = 所有方块质量之和
  - 速度 = 推力 / 质量 × 时间
  - 大气层阻力
- 飞船模块类型:
  - 框架方块 (轻甲/重甲)
  - 推进器 (化学/离子/曲速)
  - 能源核心 (裂变/聚变/反物质)
  - 功能舱 (驾驶舱/货舱/实验室)
  - 武器 (激光炮/导弹)
  - 工具 (采矿激光/牵引光束)

飞船建造模式:
- 网格对齐的体素放置
- 预览放置效果
- 碰撞检测
- 重心计算
- 飞行动力学实时更新

### 3.4 资源与科技系统

**文件**: `src/resources/resource-manager.js`
**文件**: `src/tech/tech-tree.js`

资源层级:
```
基础材料 (铁/铜/硅/碳) → 精炼材料 (钢/玻璃/塑料) → 高级零件 (电路/处理器)
                                    ↓
                            科技解锁蓝图
```

科技树结构:
- **采矿科技**: 高效采矿激光 → 自动采矿机 → 钻头阵列
- **推进科技**: 化学引擎 → 离子引擎 → 曲速引擎
- **能源科技**: 太阳能 → 裂变反应堆 → 聚变 → 反物质
- **建造科技**: 基础框架 → 真空密封 → 大气维持系统
- **防御科技**: 基础护盾 → 重型护盾 → 能量护罩

### 3.5 基地建造系统

**文件**: `src/base/base-builder.js`
**文件**: `src/base/automation.js`

功能:
- 地基铺设系统
- 墙壁/天花板搭建
- 关键建筑:
  - 发电站 (太阳能板/地热/聚变)
  - 采矿机 (自动采集)
  - 冶炼厂 (原矿→精炼)
  - 组装机 (零件制造)
  - 物流站 (物品转运)
- 传送带系统 (视觉+逻辑)
- 自动化逻辑编辑

### 3.6 探索与战斗

**文件**: `src/exploration/scanner.js`
**文件**: `src/combat/combat-system.js`

探索工具:
- 扫描仪 (分析星球成分)
- 资源定位器
- 遗迹探测器

战斗系统:
- 飞船对飞船 (炮台瞄准 + 弹药物理)
- 基地防御炮台
- 护盾系统 (能量消耗)
- 伤害计算 (装甲减伤)

### 3.7 动态事件系统

**文件**: `src/events/event-manager.js`

派系:
- 星际拓荒者联盟 (友好)
- 掠夺者舰队 (敌对)
- 神秘守望者 (环境守护者)

事件类型:
- 随机贸易船只路过
- 海盗袭击
- 守护者警告
- 遗迹发现
- 星际风暴

## 四、UI系统

**文件**: `src/ui/ui-manager.js`
**文件**: `src/ui/hud.js`
**文件**: `src/ui/menus/*.js`

界面组件:
- HUD: 资源显示、坐标、飞船状态
- 建造菜单: 分类资源/科技解锁
- 飞船仪表盘: 速度/能源/护盾/温度
- 科技树界面: 可视化节点图
- 地图界面: 星系导航
- 设置菜单: 画质/控制

## 五、控制方案

**文件**: `src/controls/input-manager.js`

支持:
- WASD移动 (飞船/探索车)
- 鼠标视角控制
- E键 交互/放置
- Q键 拆除
- B键 打开建造菜单
- T键 打开科技树
- M键 地图
- ESC 暂停/菜单

## 六、存档系统

**文件**: `src/save/save-manager.js`

存储内容:
- 飞船结构
- 基地结构
- 资源库存
- 科技进度
- 星球探索状态
- 玩家位置
